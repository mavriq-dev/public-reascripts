fft (input, inverse)
Calculates the Fast Fourier Transformation of the given input

Parameters:

  • input: A set of points that will be transformed. At this point, the input has to be a list of complex numbers, according to the format in complex.lua.
  • inverse: Boolean that controls whether a transformation or inverse transformation will be carried out.

Return value:

    Returns a list of complex numbers with the same size as the input list. Contains the fourier transformation of the input.
